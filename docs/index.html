Nothing it doesn't.

语音 AI 需要的一切，

⚡
Sentence-level Streaming TTS
句级流式 TTS
Audio starts playing before the agent finishes thinking. Each sentence is synthesized and streamed immediately — zero perceptible lag.
AI 还没回答完，声音已经开始播放。每个句子立即合成、立即播出，延迟几乎为零。
🔌
Plug-and-Play Providers
可插拔 Provider
Switch between Whisper, ElevenLabs, OpenAI, and more by changing one line in config.yaml. No code changes required.
修改 config.yaml 的一行，即可切换 ASR / Agent / TTS Provider，无需改代码。
✋
Barge-in Interruption
Barge-in 打断
Start talking mid-response to instantly interrupt the AI. Natural conversation flow — just like talking to a human.
AI 说话时直接开口，自动打断并处理你的新输入，像真实对话一样流畅。
🤖
OpenClaw-First Design
OpenClaw 优先
Built with OpenClaw as the primary agent backend. CLI and WebSocket modes supported. Also works with OpenAI, Claude, and Ollama.
以 OpenClaw 为核心 Agent，支持 CLI 和 WebSocket 两种接入模式，也兼容 OpenAI / Claude / Ollama。
🌐
Modern Web UI
现代 Web 界面
Next.js 14 frontend with push-to-talk, real-time pipeline status, streaming chat bubbles, and a settings drawer. No app install needed.
基于 Next.js 的 Web 界面，按键说话、实时状态条、流式对话气泡、设置侧边栏，浏览器直接打开。
🔒
Fully Self-Hosted
完全自托管
Your voice never leaves your server unless you choose a cloud provider. Run fully local with Whisper + Ollama + Edge-TTS.
语音数据留在你自己的服务器上。选择 Whisper + Ollama + Edge-TTS 可完全本地运行，零云端依赖。

Quick Start
快速开始

Up and running in 3 steps.

三步启动，5 分钟跑通。

1
Clone & Configure
克隆 & 配置
Copy the example config and add your API keys
复制配置模板，填入你的 API Key
2
Docker Compose Up
Docker Compose 一键启动
One command starts backend + frontend
一条命令同时启动后端和前端
3
Open & Talk
打开 & 说话
Open localhost:3000, hold Space, start talking
打开 localhost:3000，按住空格，开始说话
# 1. Clone
git clone https://github.com/tinywatermonster/listenclaw
cd listenclaw

# 2. Configure (add your API keys)
cp config.example.yaml config.yaml
nano config.yaml

# 3. Launch
docker compose up

# Open http://localhost:3000 and hold Space to talk
